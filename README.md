# AgentBook

Save web content in a format optimized for your AI agents.

## What it does

Paste any URL (X posts, YouTube videos, articles, PDFs) and get back clean, agent-readable markdown.

**Live Demo:** [agentbook.github.io](https://yourusername.github.io/agentbook)

## How to use

1. Paste a URL into the input field
2. Click "Convert"
3. Copy the formatted output for your agent

## Supported sources

- X / Twitter posts and threads
- YouTube videos (transcripts)
- Articles and blog posts
- PDFs
- Podcasts
- Any web page

## Tech stack

- Static HTML/CSS/JS
- GitHub Pages (hosting)
- Jina AI (content extraction)

## Development

```bash
# Clone and serve locally
git clone https://github.com/yourusername/agentbook.git
cd agentbook
python3 -m http.server 8080
```
